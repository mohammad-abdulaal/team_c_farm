# SEF HW8 - Laravel + React

## How to run

### backend

```sh
cd unicorns
composer install    # Generates vendor folder which contains required packages
php artisan serve   # Serves the project locally on our machine
```

### frontend

```sh
cd frontend
yarn install
expo start  # or yarn start works
```
